# 🏋️ FitLife - Fitness Website

##  Problem
Many people struggle to maintain a healthy lifestyle due to lack of proper guidance, workout plans, and tracking systems.

## 💡 Solution
FitLife is a simple fitness website that provides workout routines, diet suggestions, and motivation to help users stay fit.

## Features
- Workout Plans
- Diet Suggestions
- Daily Routine Tracker
- Responsive Design

## Tech Stack
- HTML
- CSS
- JavaScript
- Python
- SQL

## How to Run
1. Download the project
2. Open index.html in browser

##  Future Improvements
- User login system
- AI-based fitness recommendations
- Progress tracking dashboard
